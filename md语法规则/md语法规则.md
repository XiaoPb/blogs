# MD语法规则 #

## 一. 标题 ##

```c
前面带#号，后面带文字，分别表示h1-h6,只到h6，而且h1下面会有一条横线
# 一级标题 #
## 二级标题 ##
### 三级标题 ###
···
```

## 二. 列表

### 无序列表 ###

```
//形式一
+ a
+ b
+ c
//形式二
- a
- b
- c
//形式三
* a
* b
* c
```

### 有序列表 ###

```
1. abc
2. abc
3. abc
```
